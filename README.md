# Termux-SelfHosted-STT---TTS
📱 Карманный ИИ-сервер на твоем Android! Локальный Telegram-бот в Termux: переводит голос в текст (Whisper) и текст в речь (Piper). Никаких облаков — 100% приватность и вычисления на железе смартфона. Устанавливается одной curl-командой. Твой личный нейро-ассистент, который всегда под рукой! 🚀🎙️
