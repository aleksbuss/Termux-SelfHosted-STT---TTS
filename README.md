# 📱 Termux Voice AI Bot 🤖

Карманный ИИ-сервер прямо на твоем Android-смартфоне! Этот проект превращает твой телефон в полностью автономного Telegram-бота для работы с голосом и текстом. Никаких облачных API, никаких платных подписок и сливов данных — всё работает **100% локально** на процессоре твоего устройства.

## ✨ Главные фичи

- 🎙️ **Голос -> Текст (STT):** Отправь боту голосовое сообщение, и он с высокой точностью переведет его в текст с помощью whisper.cpp.
- 🔊 **Текст -> Голос (TTS):** Отправь боту текст, и он озвучит его реалистичным человеческим голосом с помощью нейросети Piper.
- 🔒 **Полная приватность:** Ни один байт ваших данных не уходит на сторонние серверы. 
- ⚡ **Установка в одну строку:** Скрипт сам скачает нужные пакеты, скомпилирует движки, загрузит модели и настроит автозапуск.
- 🔄 **Автозапуск:** Бот автоматически стартует в фоне при открытии Termux.

## 🛠 Подготовка перед установкой

Перед тем как запустить установку, вам понадобятся две вещи:

1. **Termux (обязательно из F-Droid!):** Версия из Google Play давно не обновляется. Скачайте актуальный [Termux с F-Droid](https://f-droid.org/ru/packages/com.termux/).
2. **Токен Telegram-бота:** 
   - Зайдите в Telegram и найдите бота [@BotFather](https://t.me/BotFather).
   - Напишите команду `/newbot`, придумайте имя и никнейм.
   - Скопируйте выданный HTTP API Token (он выглядит примерно так: `123456789:ABCdefGHIjklMNO...`).
